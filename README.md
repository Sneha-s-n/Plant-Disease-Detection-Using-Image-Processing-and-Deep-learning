# Plant-Disease-Detection-Using-Image-Processing-and-Deep-learning
AI-powered Plant Disease Detection using Image Processing, Deep Learning, TensorFlow, OpenCV, and Flask. Upload a plant leaf image to identify diseases accurately through a simple web interface, supporting early diagnosis and smart agriculture.

🌱 Plant Disease Detection using Image Processing and Deep Learning

This project is a web-based application that detects plant leaf diseases from images using Computer Vision and Deep Learning techniques. Users can upload a plant leaf image, and the trained model predicts the disease category with high accuracy.

✨ Features
• Plant leaf disease detection
• Image preprocessing using OpenCV
• Deep Learning (CNN/TensorFlow-Keras)
• Flask web application
• User-friendly interface
• Fast and accurate predictions

🛠️ Technologies Used
- Python
- TensorFlow & Keras
- OpenCV
- Flask
- NumPy
- HTML, CSS, JavaScript
- Bootstrap

📌 Applications
- Smart Agriculture
- Crop Disease Monitoring
- Precision Farming
- Agricultural Research
